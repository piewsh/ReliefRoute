# 🚁 ReliefRoute — Disaster Routing Optimization

An optimization framework designed for emergency medical supply distribution during disaster scenarios (e.g., Cyclone Idai). It calculates optimal routing paths to health sites based on road conditions, distance, and urgency.

### ✨ Features
- **Graph Optimization:** Uses routing algorithms to determine optimal paths between supply depots and health centers.
- **Web Interface:** Flask-based UI for visualizing routes and managing dispatch priorities.
- **Jupyter Notebook Analysis:** Contains exploratory data analysis and algorithm testing.

### 🛠️ Tech Stack
- Python, Flask
- NetworkX, Pandas, Folium (for mapping)
